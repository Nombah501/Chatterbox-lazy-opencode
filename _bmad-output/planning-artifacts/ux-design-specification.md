## Executive Summary

### Project Vision

Создать OpenCode‑first CLI/плагин, который структурирует ответы нескольких ИИ‑агентов в формат «итог + расхождения + цитаты» и при необходимости озвучивает итог через локальный TTS, снижая визуальную усталость и ускоряя выбор решения.

### Target Users

- Соло‑разработчики, проводящие много времени в CLI.
- Power users, сравнивающие ответы нескольких ассистентов.
- OSS‑мейнтейнеры, которым нужен быстрый, прозрачный синтез.

### Key Design Challenges

- Удержать формат «итог/расхождения/цитаты» читаемым и компактным в CLI.
- Обеспечить ненавязчивый TTS, не мешающий основному потоку работы.
- Дать быстрый доступ к деталям без перегрузки вывода.

### Design Opportunities

- Интерактивная навигация по секциям вывода с минимальным числом действий.
- Нативные настройки UX‑режимов: краткий/подробный, TTS on/off.
- Прозрачность синтеза через явные цитаты‑источники.

## UX Flows

### Основной поток (synthesis)

1. Пользователь запускает синтез ответов (CLI команда/флаг).
2. CLI показывает состояние обработки.
3. Выводит результат в структуре: Итог → Расхождения → Цитаты.
4. (Опционально) Озвучивает итог.
5. (Опционально) Экспортирует результат в файл.

### Настройки

1. Пользователь открывает конфигурацию.
2. Меняет TTS, голос/скорость, формат вывода, уровень детализации.
3. Сохраняет, настройки применяются глобально.

### Экспорт

1. Пользователь указывает формат (txt/md).
2. CLI сохраняет файл и сообщает путь.

## CLI Interactions & Navigation

- Режимы: синтез, настройка, экспорт.
- Навигация по секциям: явные флаги для вывода только итога/расхождений/цитат или интерактивный режим с выбором секции.
- Быстрые команды/флаги для вывода кратко/подробно.

## Output Format

**Порядок по умолчанию:**
1) Итог
2) Расхождения
3) Цитаты/источники

**Уровень подробности:**
- По умолчанию краткий итог и компактные расхождения.
- Детали включаются по запросу через флаг/интерактив.

## States

### Loading

- Короткие статусы по фазам: «Сбор ответов», «Синтез», «Формирование цитат», «Озвучка».
- В интерактиве — спиннер или прогресс‑строка.

### Error

- TTS недоступен → продолжаем без озвучки, показываем предупреждение и совет по настройке.
- Нет цитат → выводим итог/расхождения, помечаем отсутствие источников.
- Таймаут/ошибка синтеза → сохраняем частичный результат и даем действие «повторить».

## Defaults

- TTS: выключен по умолчанию, включается явно.
- Детализация: краткий режим по умолчанию.
- Экспорт: выключен по умолчанию; формат выбирается при запросе.
- Авто‑отключение TTS: включено для CI/неинтерактивных окружений.
