## Идея продукта
OSS‑инструмент для разработчиков, который синтезирует ответы нескольких ИИ‑ассистентов в OpenCode CLI и при необходимости озвучивает результат через TTS (Chatterbox как опциональный провайдер).

## Ориентир и границы
- Ориентир: AgentVibes (копируем функционал в первой версии только для OpenCode)
- Канал: гибрид CLI + плагины
- Приоритет: минимальный core

## Best practices (зафиксировано)
- Разделение результата: краткий итог + расхождения
- Прозрачность: хранить исходные ответы и цитаты‑источники
- Вывод: краткий по умолчанию, детали по запросу
- TTS: по событию (итог/алерт/подтверждение), без шума в процессе
- Формат вывода и UX: best‑practice по образцу AgentVibes

## MVP‑скоуп (черновик)
- CLI‑ядро для OpenCode
- Синтез нескольких ответов: «итог + расхождения»
- Опциональная озвучка результата через Chatterbox
- Экспорт результата в файл (txt/md)
